AMQP Server Configuration
Events are sent as AMQP messages. You can configure which server to use here.
Click on a value to edit it.
Hostname
The hostname of the AMQP server
Port
The port of the AMQP server (default 5672)
Username
The username (if any) needed for authentication
Password
The password (if any) needed for authentication
Virtual Host
The AMQP virtual host to talk to (default / )
Exchange
The AMQP exchange to use (default amqp.topic)
